About Our Learning Journey

Two friends discovering the world of web development together

Our Story

We started this journey as complete beginners, curious about how websites are built and excited to learn something new together. What began as casual interest has grown into a passionate pursuit of web development skills.


By working on separate pages of the same website, we're learning both individual development skills and collaborative teamwork. Each page reflects our unique interests and growing abilities while maintaining a cohesive overall design.


This collaborative approach has taught us the importance of communication, planning, and supporting each other through challenges and breakthroughs.

What We're Learning

HTML Fundamentals

Semantic markup, proper document structure, accessibility, and creating meaningful content with the right HTML elements.

CSS Styling

Layout techniques, responsive design, color theory, typography, and creating visually appealing and functional interfaces.

Collaboration Skills

Version control with Git, code reviews, project planning, and effective communication in development teams.

Problem Solving

Debugging techniques, breaking down complex problems, researching solutions, and learning from mistakes.

Our Goals

Short-term Goals (Next 3 months)

  • Master HTML and CSS fundamentals
  • Build responsive, accessible websites
  • Complete our first collaborative project
  • Learn basic JavaScript interactions

Long-term Goals (Next year)

  • Learn modern JavaScript frameworks
  • Understand backend development basics
  • Contribute to open source projects
  • Build full-stack web applications
